üéØ Technical Details

BVH Structure: 72 channels per frame (Hips: position + rotation, Other joints: rotation only)

Frame Rate: 60 FPS (0.016667 seconds per frame)

Coordinate System: Y-up, positive Z forward, YXZ rotation order

Motion Source: Authentic motion capture from 100STYLE dataset
